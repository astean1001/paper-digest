ParaVul: A Parallel Large Language Model and Retrieval-Augmented Framework for Smart Contract Vulnerability Detection

✍️ Tenghui Huang, Jinbo Wen, Jiawen Kang 📅 2025-10-20 원문 보기 →

📝 요약

**한 줄 요약**: 스마트 컨트랙트 취약점 탐지를 위해 경량화된 LLM과 하이브리드 RAG 시스템을 병렬로 결합한 ParaVul 프레임워크를 제안하여 높은 정확도와 효율성을 동시에 달성

**문제 정의**:
- 기존 스마트 컨트랙트 취약점 탐지 방법(정적 분석, 형식 검증)의 높은 오탐률과 낮은 확장성 문제
- LLM 기반 접근법의 높은 추론 비용과 계산 오버헤드 문제
- 단일 모델 의존으로 인한 신뢰성 한계

**핵심 기여**:
1. **SLoRA(Sparse Low-Rank Adaptation)**: 양자화된 LoRA에 희소 행렬을 도입하여 LLM 파인튜닝의 계산 비용을 대폭 절감하면서 취약점 이해 능력 향상
2. **하이브리드 RAG 시스템**: Dense retrieval과 BM25를 결합한 검색 증강 생성으로 LLM 결과 검증 및 보완
3. **메타학습 융합 모델**: RAG 시스템과 LLM 출력을 지능적으로 결합하여 최종 탐지 결과 생성
4. **Chain-of-thought 프롬프팅**: 탐지 결과에 대한 설명 가능성 제공

**의의**:
- **방법론적 혁신**: LLM과 전통적 검색 기법의 효과적 융합으로 정확도와 효율성의 트레이드오프 해결
- **실용적 가치**: 계산 비용 절감을 통한 실제 블록체인 환경에서의 적용 가능성 확대
- **연구 연관성**: LLM과 프로그램 분석 융합 연구에서 핵심적인 병렬 처리 및 다중 모델 융합 전략 제시, 취약점 탐지 자동화의 새로운 패러다임 제안

📄 원문 초록

Smart contracts play a significant role in automating blockchain services. Nevertheless, vulnerabilities in smart contracts pose serious threats to blockchain security. Currently, traditional detection methods primarily rely on static analysis and formal verification, which can result in high false-positive rates and poor scalability. Large Language Models (LLMs) have recently made significant progress in smart contract vulnerability detection. However, they still face challenges such as high inference costs and substantial computational overhead. In this paper, we propose ParaVul, a parallel LLM and retrieval-augmented framework to improve the reliability and accuracy of smart contract vulnerability detection. Specifically, we first develop Sparse Low-Rank Adaptation (SLoRA) for LLM fine-tuning. SLoRA introduces sparsification by incorporating a sparse matrix into quantized LoRA-based LLMs, thereby reducing computational overhead and resource requirements while enhancing their ability to understand vulnerability-related issues. We then construct a vulnerability contract dataset and develop a hybrid Retrieval-Augmented Generation (RAG) system that integrates dense retrieval with Best Matching 25 (BM25), assisting in verifying the results generated by the LLM. Furthermore, we propose a meta-learning model to fuse the outputs of the RAG system and the LLM, thereby generating the final detection results.
