ContractShield: Bridging Semantic-Structural Gaps via Hierarchical Cross-Modal Fusion for Multi-Label Vulnerability Detection in Obfuscated Smart Contracts

✍️ Minh-Dai Tran-Duong, Nguyen Hai Phong, Nguyen Chi Thanh 📅 2026-04-03 원문 보기 →

📝 요약

**한 줄 요약**: 스마트 컨트랙트 난독화 공격에 대응하기 위해 의미적-구조적-시간적 특징을 계층적 크로스모달 융합으로 통합한 강건한 다중 취약점 탐지 프레임워크를 제안함.

**문제 정의**:
기존 스마트 컨트랙트 취약점 탐지 시스템들이 가짜 코드 삽입, 제어 흐름 조작 등의 난독화 기법에 취약하며, 특히 기존 멀티모달 방법들이 단순 연결(concatenation) 방식으로 특징을 융합하여 모달리티 간 상호작용을 무시하고 하나의 모달리티만 난독화되어도 탐지 성능이 급격히 저하되는 문제를 해결하고자 함.

**핵심 기여**:
1. **3단계 계층적 융합 메커니즘**: Self-attention → Cross-modal attention → Adaptive weighting을 통해 모달리티 간 의미 있는 연결과 동적 가중치 조정 구현
2. **멀티모달 특징 추출**: CodeBERT(슬라이딩 윈도우)로 의미적 의존성, xLSTM으로 opcode 시퀀스의 시간적 동역학, GATv2로 CFG의 구조적 불변성 캡처
3. **난독화 강건성**: 89% Hamming Score 달성하며 난독화 하에서도 1-3% 성능 저하만 발생

**의의**:
스마트 컨트랙트 보안 연구에서 핵심 과제인 난독화 대응 문제를 다각도로 접근한 방법론적 혁신이며, 특히 크로스모달 어텐션과 적응적 가중치 기법은 악의적 공격이 증가하는 DeFi 환경에서 실용적 가치가 높음. 연구자의 스마트 컨트랙트 보안 연구에 직접 적용 가능한 강건한 탐지 프레임워크를 제공함.

📄 원문 초록

Smart contracts are increasingly targeted by adversaries employing obfuscation techniques such as bogus code injection and control flow manipulation to evade vulnerability detection. Existing multimodal methods often process semantic, temporal, and structural features in isolation and fuse them using simple strategies such as concatenation, which neglects cross-modal interactions and weakens robustness, as obfuscation of a single modality can sharply degrade detection accuracy. To address these challenges, we propose ContractShield, a robust multimodal framework with a novel fusion mechanism that effectively correlates multiple complementary features through a three-level fusion. Self-attention first identifies patterns that indicate vulnerability within each feature space. Cross-modal attention then establishes meaningful connections between complementary signals across modalities. Then, adaptive weighting dynamically calibrates feature contributions based on their reliability under obfuscation. For feature extraction, ContractShield integrates (1) CodeBERT with a sliding window mechanism to capture semantic dependencies in source code, (2) Extended long short-term memory (xLSTM) to model temporal dynamics in opcode sequences, and (3) GATv2 to identify structural invariants in control flow graphs (CFGs) that remain stable across obfuscation. Empirical evaluation demonstrates resilience of ContractShield, achieving a 89 percentage Hamming Score with only a 1-3 percentage drop
