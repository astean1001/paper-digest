Scrambler: Mixed Boolean Arithmetic Obfuscation Tool Using E-graph and Equality Expansion

✍️ Seoksu Lee, Sangjun An, Eun-Sun Cho 📅 2026-03-04 원문 보기 →

📝 요약

**한 줄 요약**: E-graph와 Equality Expansion을 활용하여 수학적 등가성이 보장된 복잡하고 다양한 MBA(Mixed Boolean Arithmetic) 난독화 표현식을 효율적으로 생성하는 도구 'Scrambler'를 제안함.

**문제 정의**:
기존 MBA 난독화 도구들이 생성하는 표현식의 복잡성과 다양성이 제한적이고, 생성된 난독화 코드의 원본 코드와의 수학적 등가성을 보장하기 어려운 문제를 해결하고자 함.

**핵심 기여**:
- E-graph 자료구조를 활용한 MBA 난독화 프레임워크 개발
- Equality Expansion 기법을 통해 구조적으로 등가성이 보장되는 복잡한 표현식 자동 생성
- 기존 도구 대비 향상된 표현력(expressiveness)과 복잡성(complexity)을 실험적으로 입증

**의의**:
MBA 난독화 도구의 성능 향상은 역설적으로 더 강력한 난독화 해제 기법 개발의 필요성을 제기함. 본 연구는 E-graph 기반 접근법의 효과를 보여줌으로써, 향후 MBA 난독화 해제 연구에서 E-graph와 equality saturation 기법을 활용한 역방향 분석 도구 개발의 중요성과 방향성을 제시하며, 공격-방어 균형 관점에서 보안 연구의 발전에 기여함.

📄 원문 초록

We propose Scrambler, and e-graph-based MBA obfuscation tool using Equality Expansion to efficiently generate complex and diverse expressions with equivalence guaranteed by construction. Experiments show Scrambler improves existing tools in expressiveness and complexity.
