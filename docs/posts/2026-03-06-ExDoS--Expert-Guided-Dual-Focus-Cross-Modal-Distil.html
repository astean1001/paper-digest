ExDoS: Expert-Guided Dual-Focus Cross-Modal Distillation for Smart Contract Vulnerability Detection

✍️ Yifan Jia, Ye Tian, Yanbin Wang 📅 2025-09-12 원문 보기 →

📝 요약

**한 줄 요약**: 스마트 컨트랙트의 소스코드와 바이트코드 간 fine-grained 정렬을 통해 바이트코드만으로도 효과적인 취약점 탐지가 가능한 크로스 모달 증류 기법 ExDoS를 제안

**문제 정의**:
실제 배포된 스마트 컨트랙트는 대부분 closed-source이므로 바이트코드만으로 취약점을 탐지해야 하는데, 기존 방법들은 graph-level에서만 소스코드와 바이트코드를 정렬하여 세밀한 구조적/의미적 상관관계를 놓치고, 바이트코드 레벨의 정확한 취약점 패턴 정보가 부족하여 모델이 판별적 특징을 학습하기 어려움

**핵심 기여**:
1. **Dual-Attention Graph Network**: node attention aggregation 모듈을 도입하여 그래프 임베딩에서 local pattern 캡처 성능을 향상
2. **Fine-grained Cross-Modal Alignment**: 기존 소스코드 취약점 패턴을 분석하여 대응되는 바이트코드 레벨 패턴을 설계하고, 이를 바탕으로 첫 번째 vulnerability pattern annotation 데이터셋 구축
3. **Expert-Guided Distillation**: 함수 레벨의 취약점 신호를 포착할 수 있는 전문가 지식 기반 증류 프레임워크 제안

**의의**:
바이트코드 분석 연구자에게 매우 중요한 참고 자료로, 특히 소스코드 없이도 정확한 취약점 탐지가 가능한 실용적 솔루션을 제시함. 제안된 fine-grained 정렬 기법과 vulnerability pattern annotation 방법론은 바이트코드 기반 보안 분석 연구의 새로운 방향성을 제시하며, 실제 블록체인 환경에서의 보안 도구 개발에 직접 응용 가능

📄 원문 초록

The success of smart contracts has made them a target for attacks, but their closed-source nature often forces vulnerability detection to work on bytecode, which is inherently more challenging than source-code-based analysis. While recent studies try to align source and bytecode embeddings during training to transfer knowledge, current methods rely on graph-level alignment that obscures fine-grained structural and semantic correlations between the two modalities. Moreover, the absence of precise vulnerability patterns and granular annotations in bytecode leads to depriving the model of crucial supervisory signals for learning discriminant features. We propose ExDoS to transfer rich semantic knowledge from source code to bytecode, effectively supplementing the source code prior in practical settings. Specifically, we construct semantic graphs from source code and control-flow graphs from bytecode. To address obscured local signals in graph-level contract embeddings, we propose a Dual-Attention Graph Network introducing a novel node attention aggregation module to enhance local pattern capture in graph embeddings. Furthermore, by summarizing existing source code vulnerability patterns and designing a corresponding set of bytecode-level patterns for each, we construct the first dataset of vulnerability pattern annotations aligned with source code definitions to facilitate fine-grained cross-modal alignment and the capture of function-level vulnerability signals.
