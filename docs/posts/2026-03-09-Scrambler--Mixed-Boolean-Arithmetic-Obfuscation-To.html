Scrambler: Mixed Boolean Arithmetic Obfuscation Tool Using E-graph and Equality Expansion

✍️ Seoksu Lee, Sangjun An, Eun-Sun Cho 📅 2026-03-04 원문 보기 →

📝 요약

**한 줄 요약**: E-graph와 Equality Expansion을 활용하여 동치성이 보장된 복잡하고 다양한 MBA(Mixed Boolean Arithmetic) 난독화 표현식을 효율적으로 생성하는 도구 Scrambler를 제안했습니다.

**문제 정의**:
기존 MBA 난독화 도구들은 생성되는 표현식의 복잡성과 다양성이 제한적이고, 동치성 보장에 어려움이 있어 효과적인 코드 보호가 어려운 문제를 해결하고자 했습니다.

**핵심 기여**:
- E-graph 기반의 MBA 난독화 프레임워크 구현으로 표현식의 동치성을 구조적으로 보장
- Equality Expansion 기법을 통해 기존 도구 대비 더 복잡하고 다양한 난독화 표현식 생성 가능
- 실험을 통해 기존 도구들보다 표현력(expressiveness)과 복잡성 측면에서 성능 향상 입증

**의의**:
연구자의 MBA 난독화 해제 연구에 직접적으로 관련되는 최신 난독화 기술로, equality saturation 기반 접근법을 실제 보안 도구에 적용한 사례입니다. 이 도구가 생성하는 복잡한 표현식들은 기존 해제 기법들의 한계를 테스트하고 더 강력한 해제 알고리즘 개발의 필요성을 제시하며, E-graph 기술의 보안 분야 활용 가능성을 보여줍니다.

📄 원문 초록

We propose Scrambler, and e-graph-based MBA obfuscation tool using Equality Expansion to efficiently generate complex and diverse expressions with equivalence guaranteed by construction. Experiments show Scrambler improves existing tools in expressiveness and complexity.
