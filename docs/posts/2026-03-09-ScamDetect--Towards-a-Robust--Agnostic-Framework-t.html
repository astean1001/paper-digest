ScamDetect: Towards a Robust, Agnostic Framework to Uncover Threats in Smart Contracts

✍️ Pasquale De Rosa, Pascal Felber, Valerio Schiavoni 📅 2025-08-09 원문 보기 →

📝 요약

**한 줄 요약**: 스마트 컨트랙트의 악성코드 탐지를 위해 바이트코드 난독화에 강인하고 플랫폼 독립적인 프레임워크 ScamDetect를 제안한 비전 논문입니다.

**문제 정의**:
- 기존 거래 기반 위협 탐지 방법은 사용자 데이터 노출과 프라이버시 문제가 있음
- 정적 바이트코드 분석 방법도 (1) 정교한 바이트코드 난독화 기법과 (2) 블록체인 환경의 이질성으로 인한 플랫폼 종속성 문제에 직면
- 저자들의 이전 연구인 PhishingHook(90% 정확도)의 한계를 극복할 필요

**핵심 기여**:
- **ScamDetect 프레임워크**: 견고하고(robust), 모듈형(modular), 플랫폼 독립적(platform-agnostic)인 스마트 컨트랙트 악성코드 탐지 시스템
- **2단계 개발 로드맵 제시**:
1. 1단계: 제어 흐름 그래프(CFG)와 그래프 신경망(GNN)을 활용하여 난독화된 EVM 바이트코드 분석
2. 2단계: 다중 블록체인 플랫폼 지원으로 확장
- **방법론적 혁신**: GNN의 복잡한 구조 패턴 포착 능력을 활용한 난독화 대응

**의의**:
스마트 컨트랙트 보안 연구에서 중요한 패러다임 전환을 제시합니다. 특히 (1) 정적 분석의 한계를 극복하는 GNN 기반 접근법, (2) 크로스 플랫폼 호환성을 고려한 설계는 향후 블록체인 보안 도구 개발의 새로운 방향을 제시합니다. 취약점 탐지 연구자에게는 난독화 대응과 확장 가능한 프레임워크 설계에 대한 중요한 통찰을 제공합니다.

📄 원문 초록

Smart contracts have transformed decentralized finance by enabling programmable, trustless transactions. However, their widespread adoption and growing financial significance have attracted persistent and sophisticated threats, such as phishing campaigns and contract-level exploits. Traditional transaction-based threat detection methods often expose sensitive user data and interactions, raising privacy and security concerns. In response, static bytecode analysis has emerged as a proactive mitigation strategy, identifying malicious contracts before they execute harmful actions. Building on this approach, we introduced PhishingHook, the first machine-learning-based framework for detecting phishing activities in smart contracts via static bytecode and opcode analysis, achieving approximately 90% detection accuracy. Nevertheless, two pressing challenges remain: (1) the increasing use of sophisticated bytecode obfuscation techniques designed to evade static analysis, and (2) the heterogeneity of blockchain environments requiring platform-agnostic solutions. This paper presents a vision for ScamDetect (Smart Contract Agnostic Malware Detector), a robust, modular, and platform-agnostic framework for smart contract malware detection.
